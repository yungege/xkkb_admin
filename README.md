Yii-admin 是使用Yii2开发的一个后台管理系统

主要功能：RBAC
其它功能正在开发中...

安装:     
	1、下载源文件或git clone https://github.com/qiaohongbo/yii2-admin.git     
	2、composer install     
	3、common/config/main.php #配置数据库      
	4、Nginx和Apache配置，并设置hosts文件     
	数据库文件：yii2-admin.sql     
	管理员账号：admin     
	管理员密码：123456
